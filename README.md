
# Hackerearth Mishipay Backend Challenge

This project holds source code of Backend Challenge hosted on Hackerearth and organised by Mishipay







## Technology Used

- Python = 3.8.0

- Django = 4.1.4

- SQL

- SQL Connector - sqllite3





## Run Locally

Clone the project

```bash
  git clone https://github.com/rajpratyush/internet_usage
```

Go to the project directory

```bash
  cd internet_usage
```

Setup Python Virtual Environment

``` In ananconda shell 
conda create --name mishipay python==3.8.0
  
```

Install Packages

``` In ananconda shell
pip3 install -r requirements.txt
```

Install Packages

``` In ananconda shell
python app.py runserver
```
